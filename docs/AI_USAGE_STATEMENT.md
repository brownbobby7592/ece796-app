# AI Usage Statement
AI tools were used to support and accelerate the development of this smartphone application by assisting with design interpretation, architecture planning, implementation guidance, and documentation. During development, AI was used to help translate the original prototype into a native Expo/React Native app structure, suggest suitable patterns for state management, BLE integration, plotting, and file export, and provide quick explanations of trade-offs between different implementation approaches. AI was also used to speed up documentation work by helping summarize design decisions, generate source-code-based explanations, and organize development notes into report-ready documents.

The final implementation decisions, code integration, testing, and project-specific adaptations were still carried out within the actual codebase. In this way, AI served as a development support tool that improved productivity, clarified technical options, and reduced time spent on repetitive writing and planning tasks.
